K-Means Clustering for Dense Network Topology
A practical implementation of unsupervised learning using K-Means clustering to analyze dense network topologies with energy distribution, developed as part of the CMPG 313 Artificial Intelligence module.

# 🚀 Key Features
K-Means Clustering: Groups 600+ network nodes based on spatial coordinates using predefined KMeans function

Energy Analysis: Examines energy distribution across clusters and identifies critical low-energy nodes

2D Network Visualization: Visualizes cluster assignments and network structure in 2D space

3D Surface Plots: Creates interactive 3D visualizations combining spatial coordinates and energy levels

Comparative Analysis: Experiments with different k values (k=7 and k=3) to observe clustering behavior

# 🛠 Tech Stack
Algorithm: K-Means Clustering (scikit-learn)

Language: Python

Libraries: Pandas, NumPy, Matplotlib, scikit-learn

Tools: Command Line, Git & GitHub

Environment: Python 3.7+
